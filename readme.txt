Solución legal integral para tu sitio WordPress.

== Description ==

Hooma Legal es un plugin de WordPress diseñado para gestionar de manera fácil y centralizada todos los aspectos legales de tu sitio web (políticas de privacidad, términos de servicio, banner de cookies, etc.).

== Installation ==

1. Sube la carpeta `hooma-legal` al directorio `/wp-content/plugins/`.
2. Activa el plugin a través del menú 'Plugins' en WordPress.
3. Ve a la sección **Hooma Legal** en el menú de administración para configurarlo.
